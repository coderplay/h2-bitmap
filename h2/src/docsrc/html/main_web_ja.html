H2 データベース エンジン

ようこそ、無料のSQLデータベース、H2へ。H2の主な特徴は:

  • 速度がとても速く、全ての方が無料で使用でき、ソースコードが含まれています
  • Javaで書かれています; GCJ (Linux) でコンパイル可能
  • エンベッド、サーバー、クラスターモードに対応
  • JDBC、 (部分的な) ODBC API; Web クライアントアプリケーション
